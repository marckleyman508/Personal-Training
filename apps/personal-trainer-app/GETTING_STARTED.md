# Getting Started - Personal Trainer App

## Installation

### Step 1: Open Terminal
Open Command Prompt or PowerShell in the project folder.

### Step 2: Install Dependencies
Run:
```
npm install
```

This installs Express, CORS, and UUID libraries.

## Running the App

### Option 1: Click start.bat (Easiest)
Simply double-click `start.bat` - it will:
- Install dependencies if needed
- Start the server
- Open your browser automatically

### Option 2: Manual Start
```
npm start
```
Then open: http://localhost:3000

### Option 3: VS Code Debug
1. Open VS Code
2. Press `F5` or go to Run > Start Debugging
3. Select "Start App" to launch server + browser
